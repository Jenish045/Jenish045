### 🔍 Network Anomaly Detection System *(March 2026)*
> Unsupervised anomaly detection on network traffic using the KDDTrain+/KDDTest+ datasets.

- Implemented **Isolation Forest**, **DBSCAN**, and **Autoencoder** (deep learning) models
- Used Autoencoder reconstruction error as the anomaly signal
- Full pipeline: preprocessing → feature encoding → normalization → evaluation
- Compared models to minimize false positives and improve reliability

`Python` `TensorFlow/Keras` `Scikit-learn` `Pandas` `NumPy` `Matplotlib`

`Python` `Pandas` `NumPy` `Scikit-learn` `Matplotlib`

### 🏥 Hospital Management System — DBMS *(April 2025)*
> A fully relational database system modelling end-to-end hospital operations — from patient intake to billing and surgery. 

- Designed a **15-entity normalized schema** covering Doctors, Patients, Nurses, Appointments, Surgery, Billing, Insurance, Medicines, Rooms, and more
- Resolved many-to-many relationships with junction tables (Doctor–Specialization, Patient–Disease, Doctor–Patient Allocation)
- Wrote advanced **SQL queries** using CTEs, JOINs, subqueries & aggregations — e.g. matching patients to doctors by disease specialization, insurance coverage analysis, room availability by department
- Generated realistic synthetic data using **Python + Faker** scripts for all 15 entities
- Produced an **ER diagram** and full project report documenting schema design decisions

`MySQL` `Python` `Faker` `SQL` `ER Modelling` `DBMS`
